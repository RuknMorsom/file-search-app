# 🔍 الباحث الذكي - نظام البحث المتقدم في الملفات

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

تطبيق بحث متقدم للملفات على الجوال والكمبيوتر، يدعم البحث بالصورة والنص داخل الملفات و PDF.

## ✨ المميزات

- 🔍 **بحث بالصورة**: ابحث عن صور متطابقة في جهازك
- 📄 **بحث في PDF بالصورة**: ابحث عن صور داخل ملفات PDF
- 📝 **بحث بالنص**: ابحث عن نصوص في الملفات
- ⭐ **مفضلة**: حفظ الملفات المهمة
- 📊 **إحصائيات**: عرض إحصائيات الملفات
- 🌙 **وضع مظلم**: يدعم الوضع الليلي
- 📱 **متجاوب**: يعمل على جميع الأجهزة
- 🚀 **PWA**: يمكن تثبيته كتطبيق على الجوال

## 🚀 التثبيت

### الطريقة 1: استخدام المتصفح
افتح الرابط: `https://yourusername.github.io/file-search-app`

### الطريقة 2: تثبيت كتطبيق PWA
1. افتح التطبيق في Chrome
2. اضغط على "إضافة إلى الشاشة الرئيسية"

### الطريقة 3: تنزيل APK
حمل أحدث إصدار من [Releases](https://github.com/yourusername/file-search-app/releases)
